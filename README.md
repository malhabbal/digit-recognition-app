** Handwritten Digit Recognition Web Application **

Model Training: Python, TensorFlow/Keras, MNIST dataset.

Backend: Flask (REST API) – lightweight, easy to deploy.

Frontend: HTML5 Canvas + JavaScript (plain or with a touch of jQuery) for drawing digits, AJAX to send the image to the backend.

Deployment: Render, Railway, or Hugging Face Spaces (all have free tiers, support Python web apps, and can deploy from GitHub).

Version Control: Git & GitHub (with branch protection and PR workflow).

GitHub repo URL: https://github.com/malhabbal/digit-recognition-app

## Live Application
The app is deployed on Render at: https://digit-recognition-app-6aj3.onrender.com/
